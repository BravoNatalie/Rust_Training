### :joystick: How to Use

#### Requirements

To run the application you'll need:
* Rust and Cargo
* * Clone the repository:
  * ```$ git clone https://github.com/BravoNatalie/Rust_Training.git```

Now go to project folder and run the app:

```bash
$ cd snake_game
$ cargo run
```
